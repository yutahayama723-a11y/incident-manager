# Incident Manager

## 概要
運用業務でのインシデント管理を効率化するために開発予定のWebアプリです。

## 作成背景
ログ分析や障害対応業務の中で、発生時刻や対応状況の管理が煩雑だと感じたため開発を決意しました。

## 使用予定技術
- Python
- FastAPI
- SQLite
- SQLAlchemy

## 今後実装予定機能
- インシデント登録機能
- 一覧表示
- 編集・削除
- ステータス管理
- ログイン機能
